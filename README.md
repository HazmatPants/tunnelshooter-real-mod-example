# TunnelShooter: Real Example Mod

This repo contains an example mod for [tunnelshooter-real](https://github.com/HazmatPants/tunnelshooter-real)

Mods use GDScript for scripting, the language used in Godot Engine, and the language TunnelShooter: Real was written in.

# WARNING
Mods are able to:
- Run abitrary code
- Access your files (with FileAcess, DirAccess, OS)
- Make HTTP requests (with HTTPClient)
- Crash the game

If you download a mod, please review its code first.

[scripts/main.gd](main.gd) contains a lot of info for available functions and variables.
